# XT2-challenge-2-Eva-Derksen
Challenge 2 voor IxD &amp; Technology 2
Voor deze challenge moest er een geanimeerde klok gemaakt worden die op verschillende schermgroottes meeschaalt en een visueel onderscheid toont tussen de fases van de dag. 
Voor het meeschalen met verschillende schermgroottes is er gebruik gemaakt van drie CSS media queries. Voor de layout is er een CSS grid toegepast. De klok toont de actuele tijd dankzij het gebruik van javascript. Om onderscheid te maken tussen de verschillende fases van de dag die hierbij horen is er een achtergrond die met de functie nextBackground past bij het dagdeel.
